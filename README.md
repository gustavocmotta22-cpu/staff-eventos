# ⚡ Staff Eventos

Plataforma SaaS completa para gestão e escalação de profissionais em eventos gastronômicos e corporativos.

## 📋 Sobre o Projeto

O Staff Eventos permite que empresas de buffet, casas de festas, organizadores de eventos, cerimonialistas e agências de staff gerenciem seus profissionais e realizem escalas de trabalho de forma automatizada.

## 🚀 Funcionalidades

- ✅ Cadastro de empresa com CNPJ e logomarca
- ✅ Múltiplos níveis de acesso (Proprietário, Administrador, Supervisor, Operador)
- ✅ Cadastro completo de profissionais
- ✅ Agenda e criação de eventos
- ✅ Escalação de equipe com IA
- ✅ Convocação automática via WhatsApp
- ✅ Confirmação online com 1 clique
- ✅ Contratos digitais simplificados
- ✅ Substituição inteligente com 1 clique
- ✅ Check-in e Check-out por GPS e QR Code
- ✅ Selfie no check-in como prova de presença
- ✅ Gestão de pagamentos com recibo automático
- ✅ Avaliação e ranking de profissionais
- ✅ Banco de talentos com perfil público
- ✅ Relatório pós-evento automático
- ✅ Dashboard com KPIs em tempo real
- ✅ Relatórios exportáveis em PDF e Excel

## 🛠️ Tecnologias (Roadmap)

- **Frontend:** React / Next.js
- **Mobile:** React Native
- **Backend:** Node.js
- **Banco de Dados:** PostgreSQL
- **Mensageria:** WhatsApp Business API
- **IA:** Python / FastAPI
- **Hospedagem:** AWS
